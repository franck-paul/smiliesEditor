Initialisation :

Si le thème actif ne contient pas le répertoire smilies où sont rangés en théorie les émoticônes, vous pouvez le créer en cliquant sur le bouton initialiser

Nouvel émoticône :

Pour créer un nouvel émoticône, vous devez associer une chaine de caractères, par exemple : :-), à l'image de votre choix .

Image :

Les formats d'image acceptés par le formulaire sont JPG ou PNG. Le plus grand côté doit être de longueur inférieure à 24 pixels.

Vous pouvez aussi envoyer un fichier ZIP valide (contenant un fichier smilies.txt).

Barre d'outils :
Elle est composée des émoticônes visibles : .

Formulaire et commentaires

Prévisualisation :
Si l'option est cochée, les images associées aux codes des émoticônes seront automatiquement affichées.
Libellé du formulaire :
Vous pouvez définir le texte précédant la barre d'outils. Si vous laissez le texte vide, le mot Émoticônes sera alors positionné dans le champs.

Rédaction de billets

Émoticônes dans la barre d'outils :
Si vous souhaitez utiliser les émoticônes dans l'écriture de nouveaux billets, vous devez activer l'option dans les préférences utilisateur (l'option est inactive par défaut).
